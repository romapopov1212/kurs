# kurs
Курсовой проект в университете:
Lана структура данных: красно-черное дерево узлы которого хеш таблицы с методом разрешения коллизий-метод цепочек.
Есть возможно добавлять элементы, удалять, искать в диапазоне дат, вводить эту структуру.
Проект реализован на языке С++ с использованием фреймворка Qt
